# Colors

> Our colors convey our brand personality and values but also help users navigate through the interface. The right combination of colors can influence how they feel, think, and behave.

For more details, see our [Colors' guideline](https://www.decathlon.design/726f8c765/p/85c5a1-colors/b/06f55e).

## How to use

You have two options in order to use colors from Vitamin Web:

### By using [CSS Custom Properties](https://developer.mozilla.org/en-US/docs/Web/CSS/Using_CSS_custom_properties)

If you don't know CSS Custom Properties, you can see basic usage documentation [here](https://developer.mozilla.org/en-US/docs/Web/CSS/Using_CSS_custom_properties#Basic_usage). All our CSS Custom Properties for colors are listed after in our palette at "Swatches" column.

### By using [Tailwind CSS](https://tailwindcss.com) utility classes

You can use colors with Tailwind CSS utility classes for [Background Color](https://tailwindcss.com/docs/background-color) & [Text Color](https://tailwindcss.com/docs/text-color) for example.
But you can also use them for borders, gradients, etc. Go to [Tailwind CSS documentation](https://tailwindcss.com/docs) & quick search for anything!
All our Tailwind CSS names for colors are listed after in our palette at "Name" column.
